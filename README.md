## 下载

### Docker

```
docker pull apachecn0/dsai-conf-notes
docker run -tid -p <port>:80 apachecn0/dsai-conf-notes
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dsai-conf-notes
dsai-conf-notes <port>
# 访问 http://localhost:{port} 查看文档
```
